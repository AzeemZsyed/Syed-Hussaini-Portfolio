 # Syed Hussaini — Personal Portfolio

### A minimal, space-themed developer portfolio built from scratch with pure HTML, CSS, and JavaScript.

A dark, star-field portfolio site showcasing my experience, projects, and contact info as a Software Engineering Co-op student at the University of Guelph.

---

## Overview

This portfolio is a multi-page static site with a shared aesthetic across all pages — animated star background, outline typography, and a fixed bottom navigation bar with per-link hover colors.

```
Portfolio/
│
├── Home.html          ← Landing page with animated hero
├── Experience.html    ← Work history
├── Projects.html      ← Project showcase
├── LinkedIn.html      ← Contact & links
├── Resume.html        ← Resume download
│
├── Style.css          ← All shared styles
└── Background.js      ← Star animation (shared across pages)
```

---

## Page Flow

```
Home → Experience → Projects → LinkedIn → Resume
 ↑___________________________________________|
              (bottom nav bar)
```

---

## Tech

- HTML5 / CSS3 / Vanilla JavaScript
- No frameworks, no dependencies
- Responsive bottom nav with per-link accent colors
- Animated star field using JS-generated `box-shadow` particles

---

## Run Locally

No build step needed. Clone the repo and open `Home.html` in your browser:

```bash
git clone https://github.com/AzeemZsyed/Syed-Hussaini-Portfolio.git
cd Syed-Hussaini-Portfolio
open Home.html
```
